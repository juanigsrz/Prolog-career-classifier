    INTRODUCCION

El objetivo de este trabajo es construir un pequeño sistema experto utilizando
programación lógica que implemente un test vocacional.

    CLASIFICACION

Nuestro sistema procede a atacar el problema de clasificación en 3 etapas:

* ÁMBITOS       Una primera aproximación al área de interés del usuario.
* RASGOS        Se desprenden de las áreas de interés y describen las aptitudes
                del usuario.
* ORIENTACIONES Secciones que agrupan dos o tres carreras, y corresponden a la
                clasificación más fina antes de las carreras en sí.
